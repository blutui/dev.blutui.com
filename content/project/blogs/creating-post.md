## Create a New Post

You can create a post by clicking on the `Create post` button on the top right hand side of the page. You will be put into the blog `editor`.

![Blog Creating Post ](https://cdn.blutui.com/uploads/assets/Help/Site-Dashboard/blogs/blog-2023-5.png)

### Adding Content

You can add a `Title`, `Excerpt` and `Content`. You will be able to add all the content into the `Content` section including `images`, `lists` and more.

### Publish Date

You can set a publish time of the post. If you do set this into the future it will not show up on your site.

### Author

You can add a name to who wrote the post here. It will default to the account name your using to write the blog.

### Feature Image

You can upload or search in your `assets` for a featured image for this post.

### Permalink

you can set the url link for the post. If you dont it will generate one automatically.
