## Create a New Page

You can create a page by clicking on the `Add page` button on the top right hand side of the page.

## Managing a Page

![Page management](https://cdn.blutui.com/uploads/assets/Help/Site-Dashboard/pages/mange-page.png?format=webp&width=800)

### Editing Page details

You can change various parameters of your page such as the `Name`, `URL`, `Meta title`, `Meta description`, `Visibility` , `Layout` and the `Publish Date`.

#### Name

You can set your page `Name` here.

#### URL

You can set your `URL` here.

#### Meta title

You can set your `Meta title` here.

#### Meta description

You can set your `Meta description` here.

#### Visibility

You can set your page to be `visible` or `hidden` when `hidden` is select your page won't be visible if deployed.

#### Layout

You can select the template for this page.

#### Publish Date

Schedule a date for this page to become public.
